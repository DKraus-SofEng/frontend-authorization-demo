# Cryptoducks

A React app built with Vite. Used by the TripleTen Software Engineering Bootcamp to demonstrate frontend authorization and authentication.

## Usage

1. **Clone the repo:**
   ```bash
   git clone <repo-url>
   cd frontend-authorization-demo
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open the app:**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- User registration and login
- JWT-based authentication
- Protected routes using React Context and custom route guards
- Logout functionality
- Modern React (hooks, context, Vite)

## Branches

- `main`: Starting code
- Other branches: Solution code for each lesson

## Notes

- The app uses a public Strapi backend for demonstration. Some features may be limited due to backend restrictions.
